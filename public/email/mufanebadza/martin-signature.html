Installing in Zoho Mail

Prime Cane mail runs on Zoho. Use the Insert HTML option in the Zoho signature editor — it takes the raw markup without breaking the table layout.

Zoho Mail
  1. Sign in to mail.zoho.com as martinm@primecane.com.
  2. Click Copy HTML code on the card above (Hosted URLs mode).
  3. Settings → Signatures → + to add one; name it Prime Cane.
  4. In the editor toolbar open Insert HTML (</> icon) and paste the code.
  5. Use Associate Mail Address to link it to the From address → Save.
Gmail / Apple Mail
  1. Click Copy signature instead — it pastes the finished design.
  2. Gmail: Settings → Signature → Create new, then paste.
  3. Apple Mail: Settings → Signatures, untick Match default font, then paste.
Outlook
  1. New Outlook / web: paste with Copy signature under Compose and reply.
  2. Classic Outlook: use Copy HTML code, save as a .htm file in the Signatures folder.
  3. Always use Hosted URLs for Outlook.
