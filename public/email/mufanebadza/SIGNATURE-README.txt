HOW TO USE
----------

1. Open primecane-email-signature.html in a web browser (Chrome recommended).
2. Choose an image source at the top:
     - Hosted URLs  -> recommended for Zoho & Outlook; needs the
                       email-images uploaded to a public URL (see below)
     - Embedded     -> self-contained, no hosting; Zoho may strip large
                       embedded images
3. Click "Copy HTML code" (for Zoho "Insert HTML") or
   "Copy signature" (for Gmail / Apple Mail).
4. Paste into your mail client's signature editor.


HOSTING THE IMAGES
------------------

The signature expects the images at:

  https://primecane.com/email/<filename>

Upload everything in the "email-images" folder there once the domain is
live. Until then, host them on any public bucket and find-and-replace
that base URL in the copied HTML code.


ZOHO MAIL INSTALL
-----------------

1. Sign in to mail.zoho.com as martinm@primecane.com.
2. In the kit, choose "Hosted URLs", then "Copy HTML code".
3. Settings -> Signatures -> "+" to add a new signature; name it Prime Cane.
4. In the editor toolbar, open "Insert HTML" (the </> icon) and paste.
5. Use "Associate Mail Address" to link it to the From address. Save.
6. Send yourself a test email to confirm it renders.
